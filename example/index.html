Select Drop

Simple jQuery plugin to convert select to dropdown

See on Github Download

Basic example:

$(document).ready(function() {
  $('#first-select').selectDrop()
})

Options:

Your choice: "First"

$(document).ready(function() {
  $('#second-select').selectDrop({
      eventType: 'hover'   // you can change the event
    , onChange: callback   // and trigger callback functions on change
    , autoSubmit: false    // if set to true, the parent form will be
                           // submitted when the value is changed
  })
})

function callback() {
    $('#choice').text($('#second-select').val())
}
